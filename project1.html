Smart Electricity Saver

This project is designed to reduce electricity wastage by automatically controlling power using motion sensors and smart logic.

🔧 Technologies Used:

  • Arduino
  • Motion Sensors
  • Basic AI Logic

⚡ Features:

  • Automatic ON/OFF system
  • Energy efficient
  • Low-cost solution

🌍 Impact:

This system can help reduce electricity bills and is especially useful for low-income households and schools.
